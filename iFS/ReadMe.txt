iFS 1.0
------------------------------------
Позволяет работать с фалами на устройствах iOS
Allows managing files on iOS devices
------------------------------------

Распакуйте zip в папку Plugins FAR Manager.
Запустите install.cmd от имени администратора
(в контекстном меню файла выберите "Запуск от имени администратора")
Это необходимо для регистрации COM-библиотеки idevice.
После этого плагин готов к работе.
Подключенные устройства будут появляться в меню выбора диска.

Unpack zip to FAR Manager Plugins directory.
Run install.cmd as Administrator
(select "Run as Administrator" from context menu).
This is necessary in order to register idevice COM-library.
After that plugin is ready to use.
Connected devices will appear in disk selection menu.
