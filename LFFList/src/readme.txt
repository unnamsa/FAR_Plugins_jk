Для сборки динамических библиотек плагина вам понядобится:

1. GIT Клиент (я брал отсюда: https://git-scm.com/download/win).

2. Сборка MinGW, поддерживающая сборку 64х разрядных бинарников, я
   взял сборку отсюда:
   (http://mingw-w64.sourceforge.net/download.php#mingw-builds).

   При установке указанной мною сборки необходимо для опции  Exception
   выирать значение "sjlj", все остальные настройки можно оставить  по
   умолчанию,  тогда  будет  установлена  x32  версия  компилятора   с
   возможностью  сборки  x64  бинарников.  После  чего  в  подкаталоге
   mingw32\bin   каталога   MinGW    необходимо    скопировать    файл
   mingw32-make.exe в make.exe.


3. В PATH должен быть добавлен путь до исполняемых файлов MinGW.

   Что бы собрать динамические библиотеки нужно запустить файл
build.bat, при этом разово будут загружены исходники Far Manager в
подкаталог !far_src каталога где лежит плагин.
